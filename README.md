# multi-menu

Aplicação PHP simples para gerenciar múltiplos cardápios.

## Configuração

1. Aponte o **DocumentRoot** do servidor web para o diretório `public`.
2. Certifique-se de que o `mod_rewrite` esteja habilitado e que o arquivo `.htaccess`
   incluído redirecione todas as requisições para `index.php`.

O sistema agora detecta automaticamente o caminho base, permitindo a execução em
qualquer subdiretório sem ajustes adicionais.
